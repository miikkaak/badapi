# Bad API

This is a badly written simple NodeJS/Express REST API to demonstrate the *smsbrute.py* script found in [HERE](https://github.com/miikkaak/smsbrute)

Run with
```
npm start
```

## Methods

1. **"/" (GET)**
    - Display default message
2. **"/gen" (GET)**
    - Generate new codes to crack
3. **"/" (POST)**
    - Bombard this
